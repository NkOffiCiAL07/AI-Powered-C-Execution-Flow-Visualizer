📋 WHAT YOU NEED:
═════════════════════════════════════════════════════════════════════════

✓ Python 3.10+
✓ Node.js 16+
✓ LLDB or GDB installed
  (macOS: comes with Xcode)
  (Linux: apt-get install lldb)


🚀 3-STEP STARTUP:
═════════════════════════════════════════════════════════════════════════

STEP 1: Start the Backend Server
─────────────────────────────────
  cd /path/to/AI-Powered-C-Execution-Flow-Visualizer
  python3 run_server.py

  ✓ Server starts on http://localhost:8000
  ✓ You'll see: "Uvicorn running on http://127.0.0.1:8000"


STEP 2: Install Frontend Dependencies (first time only)
─────────────────────────────────────────────────────────
  cd frontend
  npm install

  ✓ This takes 1-2 minutes


STEP 3: Start the Frontend Dev Server
──────────────────────────────────────
  npm start

  ✓ Browser opens automatically to http://localhost:3000
  ✓ You're ready to go!


💡 QUICK TEST:
═════════════════════════════════════════════════════════════════════════

1. Paste this code in the editor:

   #include <iostream>
   int main() {
       int x = 5;
       int y = 10;
       int sum = x + y;
       std::cout << sum << std::endl;
       return 0;
   }

2. Click "Analyze & Run"

3. Watch the animation:
   ✓ x changes from ? → 5 (flashes orange)
   ✓ y changes from ? → 10 (flashes orange)
   ✓ sum changes from ? → 15 (flashes orange)

4. Use controls:
   ⏮️ Restart | ▶️ Play/Pause | ⬅️ ➡️ Step | Speed 🎚️


🎮 CONTROLS:
═════════════════════════════════════════════════════════════════════════

Buttons:
  ⏮️  Restart   → Go back to step 1
  ▶️  Play     → Start auto-animation (changes to ⏸️ Pause)
  ⏸️  Pause    → Stop animation
  ⬅️  Previous → Go to previous step
  Next ➡️      → Go to next step

Slider:
  🎚️ Speed Control
     Left  = Faster (20x)
     Right = Slower (0.5x)

Tabs:
  📊 Execution Flow → Animated visualization (default)
  📋 Output & Details → Detailed table view


📊 WHAT YOU SEE:
═════════════════════════════════════════════════════════════════════════

When a variable changes:
  🟠 Orange highlight    → Variable just changed
  ⚡ Flash animation     → Attention-grabbing effect
  ← Arrow with old value → Shows what it was before
  ⭐ "NEW" badge         → Pulsing indicator

Timeline:
  🔵 Blue pulsing dot    → Current step
  🟢 Green dots          → Already executed
  ⚪ Gray dots           → Future steps
  Click any dot to jump to that step

🔍 TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════

Problem: "Cannot connect to server"
  Solution: Make sure backend is running on terminal 1

Problem: "Code won't compile"
  Solution: Use valid C++ code with #include statements

Problem: "Frontend won't load"
  Solution: npm install was successful? Try: npm start again

Problem: "Variables not showing"
  Solution: Try this simple example:
    int x = 5;
    int y = x + 10;
    int z = y * 2;

Problem: "Animation is too slow"
  Solution: Move speed slider to the left (faster)
